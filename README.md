# WoofFlow-AI-Email-Booking-Automation-Prototype
AI-powered workflow that automates customer booking email booking requests using NLP, reducing manual effort and improving response efficiency.
# 🧩 Project Title
WoofFlow: AI Email Booking Automation System
# 📌 Problem Statement
Businesses handling customer bookings email via email face delays, manual effort, and inconsistent responses. This leads to poor customer experience and operational inefficiencies.

# 🎯 Objective
- Automate email classification and booking extraction
- Reduce response time
- Improve accuracy of booking handling

# 🛠️ Tools & Technologies
- n8n (workflow automation)
- Gmail API
- OpenAI Models(NLP)
- Google Sheets
- JSON Processing

# 🔍 Approach
- Built an automated pipeline to ingest incoming emails
- Classified email intent using NLP models
- Extracted booking details (data,time, service)
- Generated automated response drafts
- Stored structured data in Google Sheets

# 📊 Key Features
- End-to- end automation pipeline
- Real-time email processing
- Intelligent intent classification
- Structured data extraction

# 📈 Results / Insights
- Reduced manual email handling effort significantly
- Improved response consistency
- Enabled scalable customer communication

# 🚀 Business Impact
This system can help service-based businesses scale operations without increasing customer support costs.
